**Course:** ITAI 4374  
**Modules:** 06 & 07  
**AI System Tested:** Claude (claude.ai)  
**Tool:** Jupyter Notebook + Claude

---

## What This Lab Was About

This lab was a practice audit — testing a real AI system (Claude) through the lens of what we learned about memory and attention in the brain. The goal was to run structured experiments, record actual responses, and fill out a Brain vs. AI Scorecard comparing Claude's behavior to biological memory and attention systems.

---

## What I Tested

**Part A — AI Memory (Module 06)**
Tested whether Claude could retain and recall information across a conversation, similar to how working memory and the hippocampus handle short-term and long-term storage in the brain.

**Part B — AI Attention (Module 07)**
Used Google Colab for attention visualizations, then tested whether Claude could focus on relevant information while filtering out distractions — comparing it to how the brain's frontoparietal attention network works.

**Part C — Audit Report**
Wrote up findings in a structured Brain vs. AI Scorecard format, rating Claude's performance on each dimension and explaining the differences from biological systems.

**Part D — Midterm Connection**
Used this lab as direct preparation for the midterm group audit project.

---

## What I Learned

Testing an AI system against actual neuroscience criteria is a completely different experience from just using it. You notice things you'd normally ignore — like how Claude handles a question after several unrelated messages, or what happens when you give it competing instructions. The scorecard format forced me to be specific instead of just saying something "worked" or "didn't work." That structure turned out to be the most useful part of the lab.

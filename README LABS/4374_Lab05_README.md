## What This Lab Was About

This lab was about how the brain turns raw physical signals — light, sound, pressure — into the experience of seeing, hearing, and feeling. The idea was to apply the same computational principles the brain uses for sensory processing, then compare that to how artificial neural networks learn to do the same thing on their own.

---

## What I Learned

The brain doesn't just passively receive information from the senses. It actively transforms raw input through a series of processing stages — filtering, detecting edges, identifying patterns — before anything reaches conscious awareness. What's interesting is that CNNs essentially rediscovered this approach through training. Early layers in a CNN learn edge detectors, and later layers learn more complex shapes, which closely mirrors how the visual cortex is organized.

This lab made that connection feel real instead of just theoretical. Doing the processing yourself and then seeing how a neural network arrives at similar solutions on its own was a useful way to understand why brain-inspired architectures work.
